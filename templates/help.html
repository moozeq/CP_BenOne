BeNone is an web service for examine your data if it's complaint to Benford's Law. What is Benford's Law you may ask? Check it out here!

First you need to Upload your file to our servers or just use someone's else file. File should be encoded with utf-8 in one of following formats:
 - .csv (delimiter = ',')
 - .tsv (delimiter = '\t')

When upload is completed, select file (if you've uploaded it, it'll be selected automatically), choose it's type between available ones and press Analyze.

After successful analysis, you will be prompted to select column (based on headers names) from your data which will be compared against Benford's Law digits distribution.

When selected column, on graph you should see your data and below graph there will be information about Benford's Law data compliance, one of following prompts:

Benford's law compliant: 13.37% (Two-sample Kolmogorov-Smirnov test p-value = 0.1337) Benford's law compliant: 95.37% (Two-sample Kolmogorov-Smirnov test p-value = 0.9537)

Below that prompt there are some stats from analysis of selected file, pretty self-explanatory, so i.e. your data file is corrupted and some lines aren't corresponding to header, they will be omitted and count of those lines will be prompted to you in this section. Example: 


{
  "benford": {                      # columns benford's law compliances
    "3": 0.0007,
    "4": 0.3517,
    "7_2009": 1,
    "8.40188": 0.0336,
    "State": 0,
    "Town": 0
  },
  "ext": ".tsv",                    # extension selected by user
  "filename": "census_2009b",       # filename
  "hash": "16032f...7356.tsv",      # file content hash + its extension
  "header_size": 6,                 # size of header elements
  "omitted_lines": 6,               # lines which couldn't be parsed
  "parsed_lines": 19509,            # lines properly parsed
  "parsed_words": 117018            # count of parsed words in file
}
